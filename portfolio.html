Projects

Getta Gif

Simple App that uses the Gyphy API to display gif's on the screen based on the search button pressed. Also allows the user to make thier own search button.

Crystal Collector

Math game that uses random numbers to determine the target score, and also changes the values of each gem so that the user has to develop a strategy to reach the target score without going over.

Marvel Trivia Game

Trivia game based on the Marvel Cinematic Universe. Each question is timed and the user must click on an answer within the time alloted or it counts an incorrect.

Train Time

Train scheduling application that utilizes Firebase to store the input and display the information on the page. Also uses the moment.js library to calculate arrival times.

Orange Swan Planning

Orlando based travel planning app that will help find flights to Orlando, and also lists dining, lodging, and theme park recommendations. Uses the Skyscanner API and Google Places API.

liri_node_app

Command line application that takes in a command and a search term, then prints out data from the Spotify, Bands In Town, or Open Movie Database API.
